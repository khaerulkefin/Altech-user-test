#SWEETESCAPE.COM
#Selenium java automation

#Description
This is an automation project using Selenium. In this project I use Java language. The purpose of this project is to ensure that the features used are appropriate, especially the register, login and order features.

#Features
- **Register** : Memastikan user dapat membuat akun baru
- **Login** : Memastikan user dapat melakukan login dengan email yang valid
- **Order** : Memastikan user dapat melakukan booking photoshoot

#Pre-requisites
- **java version "21.0.4"** or latest
- **Maven**
- **Eclipse**
- **Webdriver**
